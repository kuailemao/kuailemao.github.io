🍟8.2、装箱拆箱

Java 基本数据类型变量和包装类对象之间的转换较繁琐，从 JDK1.5 之后提供了自动装箱（Autoboxing）和自动拆箱（AutoUnboxing）机制

  • 自动装箱：把基本数据类型变量直接转换为对应的包装类对象，或者转换为 Object 对象。
  • 自动拆箱：与装箱相反，将包装类对象转换成对应的基本数据类型变量
